Drizzle
The simplest jQuery filtering and sorting library ever!

The main idea behind drizzle is to keep things simple and avoid complex conditions, thus evading confusion. No more time-wasting. Let's get started with it.
Including drizzle
The first thing to do is to include the plugin file to your html.

To include the plugin file, you can either download it include it locally or use CDN path

                    
                        <script type="text/javascript" src="./js/drizzle.min.js"></script>
                        <!-- OR -->
                        <script type="text/javascript" src="https://cdn.jsdelivr.net/gh/anishnair02/drizzle/dist/drizzle.min.js"></script>
Pros and cons of using local vs CDN file
  • Downloading and using the plugin file locally in your project is faster.
  • But using CDN path has a great advantage - you get the latest copy of the JS file.
    Also note, browser caching or server side caching may affect fetching the latest version of drizzle.min.js
    since the filename remains the same and at times maybe cached.
Initializing drizzle
Once you have included the plugin file, you need initialize it once (for a particluar parent-child element relationship) to be able to filter and sort elements.
Go to your custom js file or create a new one - say "main.js" and include it to your html file just like you did above.

To initialize, please refer to the code below

                    
                        var grid = $(container).drizzle({child:child elements});

Here, we are defining the container/parent element and the child elements so that drizzle understands it.

Container and child elements can be any HTML/jQuery element(i.e a jQuery selector) and you can define it using an id, classname or by using any attribute-value pair

#id .classname [attribute=value]

Example
                        
                            var grid = $('.cards').drizzle({child: '.card'});
Filtering
To filter the children, use the filter method
                    
                        grid.filter(element);

Element can be any HTML/jQuery element(i.e a jQuery selector) and you can define it using an id, classname or by using any attribute-value pair

#id .classname [attribute=value]

Example
                        
                            grid.filter('.members.new');
                            /* OR */
                            grid.filter('#customer-21');
                            /* OR */
                            grid.filter('[data-role="captain"]');
Sorting
To sort the children, use the sort method
                    
                        grid.sort(element, [options]);

Element can be any HTML/jQuery element(i.e a jQuery selector) and you can define it using an id, classname or by using any attribute-value pair

#id .classname [attribute=value]

Example
                        
                           grid.sort('.content .members.new', {order: 'desc'});
Unfilter or Reset
To reset or unfilter, use any of the below methods
                    
                        grid.unfilter();
                        /* OR */
                        grid.rain();
                        /* OR */
                        grid.filter('*')
